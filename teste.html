Técnicas para Ofuscar Imagens

Explore diferentes métodos CSS para ofuscar, esconder parcialmente ou criar efeitos visuais em imagens

Efeito Blur

Desfoca a imagem usando o filtro CSS blur().

filter: blur(5px);

Redução de Brilho

Escurece a imagem usando o filtro brightness().

filter: brightness(0.3);

Redução de Contraste

Diminui o contraste da imagem usando o filtro contrast().

filter: contrast(0.2);

Overlay Escuro

Adiciona uma camada semi-transparente sobre a imagem.

.overlay::after {
  content: '';
  position: absolute;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background: rgba(0, 0, 0, 0.6);
}

Overlay com Gradiente

Adiciona um gradiente escuro sobre a imagem.

background: linear-gradient(to bottom,
  rgba(0,0,0,0.2) 0%,
  rgba(0,0,0,0.8) 100%
);

Efeito Pixelado

Cria um efeito de pixelização na imagem.

image-rendering: pixelated;
transform: scale(1.5);

Combinação de Efeitos

Combina múltiplos filtros para um efeito mais complexo.

filter: blur(3px) grayscale(0.7) opacity(0.8);
